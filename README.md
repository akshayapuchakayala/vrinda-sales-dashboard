# Vrinda Store Sales Dashboard

## Project Overview
This project analyzes Vrinda Store sales data using Python, SQL, and Power BI.  
The objective was to clean raw sales data, perform business analysis, and build an interactive dashboard to generate meaningful business insights.

---

## Tools & Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- MySQL
- Power BI
- Jupyter Notebook

---

## Key Skills Demonstrated
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- KPI Metrics Calculation
- SQL Query Writing
- Data Visualization
- Dashboard Design
- Business Insight Generation

---

## Data Cleaning Performed
- Removed duplicate records
- Standardized gender values
- Converted date columns into proper format
- Created age groups
- Handled missing values
- Renamed columns for analysis

---

## SQL Analysis
Performed SQL queries to analyze:
- Total Revenue
- Monthly Revenue Trend
- Revenue by State
- Revenue by Category
- Revenue by Sales Channel
- Order Status Analysis

---

## Power BI Dashboard Features
- KPI Cards
- Monthly Revenue Trend
- Channel-wise Revenue Analysis
- State-wise Revenue Analysis
- Category-wise Revenue Analysis
- Gender & Age Group Analysis
- Interactive Slicers & Filters

---

## Key Business Insights
- Women customers generated higher revenue than men
- Amazon was the top-performing sales channel
- Maharashtra contributed the highest revenue
- Set category generated maximum sales
- Most orders were successfully delivered

---

## Dashboard KPIs
| KPI | Value |
|---|---|
